# Migrate from Hindsight

Mnemosyne v3.7.0 includes a **Hindsight importer** that migrates memories from Hindsight self-hosted into Mnemosyne's episodic storage. Unlike the generic import path (which routes through `remember()` and assigns current timestamps), the Hindsight importer writes directly to `episodic_memory` — preserving the full historical record.

## What Gets Preserved

| Hindsight Field | Mnemosyne Destination |
|-----------------|------------------------|
| `mentioned_at` | `timestamp` (original value) |
| `fact_type` | `source` as `hindsight:{fact_type}` |
| `tags` | `session_id` (session tags extracted) |
| `chunk_id` | Stable hash in `session_id` |
| `proof_count` | `importance` score bonus |
| `metadata`, `context`, `entities` | `metadata_json` (full preservation) |
| `id` | `hindsight_id` in metadata |

## Input Sources

The importer accepts two input sources:

1. **JSON export file** — a Hindsight export containing memories
2. **Live HTTP API** — a running Hindsight instance at `/v1/default/banks/{bank}/memories/list`

## CLI Usage

```bash
# From JSON export
mnemosyne import-hindsight hindsight-export.json hermes

# From live API
mnemosyne import-hindsight http://localhost:8888 hermes
```

## Python API

```python
from mnemosyne.core.importers import HindsightImporter, import_from_hindsight

# From JSON export
importer = HindsightImporter(file_path="hindsight-export.json", bank="hermes")
result = importer.run(mnemosyne)

# From live API
importer = HindsightImporter(base_url="http://localhost:8888", bank="hermes")
result = importer.run(mnemosyne)

# Convenience wrapper
result = import_from_hindsight(mnemosyne, file_path="export.json", bank="hermes")
```

## Result Object

```python
result.provider      # "hindsight"
result.total         # Total items found
result.imported      # Successfully inserted
result.skipped       # Duplicates or empty content
result.failed        # Insertion errors
result.memory_ids    # List of imported memory IDs
result.errors        # List of error strings
```

## Duplicate Handling

The importer uses **stable SHA256-based IDs** with an `hs_` prefix. Re-importing the same export is idempotent — duplicates are skipped via `INSERT OR IGNORE`.

## Importance Scoring

Hindsight `fact_type` maps to base importance:

| `fact_type` | Base Importance |
|-------------|-----------------|
| `world` | 0.75 |
| `experience` | 0.65 |
| `observation` | 0.55 |
| (other) | 0.50 |

`proof_count` adds up to 0.15 bonus (capped at 1.0).

## Provider Registry

Hindsight is registered as the 7th supported provider:

```python
from mnemosyne.core.importers import import_from_provider, PROVIDER_REGISTRY

# Generic dispatcher
result = import_from_provider("hindsight", mnemosyne, file_path="export.json")
```
