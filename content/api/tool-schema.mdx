# Tool Schema

Complete JSON Schema for Mnemosyne tools exposed to agents.

## Tool Selection Guide

| Goal | Tool | Example |
|------|------|---------|
| Store general info | `remember` | User preferences, decisions |
| Store relational facts | `add_fact` | Team structure, dependencies |
| Retrieve context | `recall` | "What did we decide?" |
| Precise lookup | `query_facts` | "Who reports to Carol?" |
| Remove bad data | `forget` | Outdated information |

<Callout type="success" title="Schema Validation">
Tool calls are validated against these schemas before execution. Invalid parameters return a structured error that the agent can use to self-correct.
</Callout>
