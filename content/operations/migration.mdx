# Migration

Upgrade Mnemosyne versions and migrate data between instances.

## Version Upgrade

### Check Current Version

<CodeBlock code={`python -c "import mnemosyne; print(mnemosyne.__version__)"
`} language="bash" />

### Upgrade Steps

<CodeBlock code={`# 1. Backup current database
cp mnemosyne.db mnemosyne-pre-upgrade.db

# 2. Upgrade package
pip install --upgrade mnemosyne-memory

# 3. Verify (schema migrations run automatically on first connection)
python -c "from mnemosyne import Mnemosyne; m = Mnemosyne(); print(m.stats())"
`} language="bash" />

<Callout type="info" title="Automatic Migrations">
Mnemosyne uses automatic schema migrations that run on first connection after an upgrade. There is no `python -m mnemosyne.migrate` command. If migration fails, enable debug logging with `logging.getLogger("mnemosyne").setLevel(logging.DEBUG)`.
</Callout>

## Schema Migrations

Mnemosyne uses automatic schema migrations via lazy `ALTER TABLE ... ADD COLUMN` on first connection. When the codebase adds new columns to the schema, `init_beam()` checks whether each column exists and runs `_add_column_if_missing()` as needed. No manual migration commands or version-specific migration functions are required.

## Cross-Instance Migration

Export from old instance, import to new using the Python SDK:

<CodeBlock code={`from mnemosyne import Mnemosyne

# Export from old database
old_mem = Mnemosyne(db_path="old.db")
old_mem.export_to_file("export.json")

# Import into new database
new_mem = Mnemosyne(db_path="new.db")
new_mem.import_from_file("export.json")

# Verify
print(new_mem.stats())
`} language="python" />

### Export Formats

| Format | Use Case | Size |
|--------|----------|------|
| JSON | Human-readable, version control | Large |
| SQLite | Fast, preserves all metadata | Compact |

## Legacy Migration

For migrating from pre-2.0 data formats, use the migration script:

<CodeBlock code={`python scripts/migrate_from_legacy.py --db mnemosyne.db
`} language="bash" />

## Rollback

If upgrade fails:

<CodeBlock code={`# Stop application
# Restore pre-upgrade backup
cp mnemosyne-pre-upgrade.db mnemosyne.db

# Downgrade package
pip install mnemosyne-memory==1.0.0

# Restart
`} language="bash" />

<Callout type="warning" title="Breaking Changes">
Major version upgrades (1.x → 2.x) may include breaking changes. Always read the changelog and test in staging before production deployment.
</Callout>
