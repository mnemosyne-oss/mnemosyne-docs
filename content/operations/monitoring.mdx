## Key Metrics

| Metric | Target | Alert If |
|--------|--------|----------|
| Query latency p50 | &lt;100ms | &gt;200ms |
| Query latency p99 | &lt;500ms | &gt;1000ms |
| DB size | &lt;1GB | &gt;5GB |
| Working memory | &lt;80% max | &gt;95% |
| Last consolidation | &lt;2x interval | &gt;3x interval |

## Alerting Rules (Prometheus)

<CodeBlock code={`groups:
  - name: mnemosyne
    rules:
      - alert: HighQueryLatency
        expr: mnemosyne_query_latency_ms{quantile="0.99"} > 500
        for: 5m
        annotations:
          summary: "Mnemosyne query latency is high"

      - alert: MemoryNotConsolidating
        expr: time() - mnemosyne_consolidation_last_run > 7200
        for: 10m
        annotations:
          summary: "Mnemosyne consolidation has stalled"
`} language="yaml" />
