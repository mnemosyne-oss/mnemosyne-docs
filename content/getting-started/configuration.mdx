# Configuration

Mnemosyne is configured via environment variables and (in Hermes Agent) through `config.yaml`. All settings have sensible defaults so you can get started with zero configuration.

## Constructor Parameters

The only runtime parameters are passed directly to the `Mnemosyne()` constructor:

```python
from mnemosyne import Mnemosyne

mem = Mnemosyne(
    session_id="my-agent",   # Scope working memories to this session
    db_path="./data/mem.db", # Custom database path (optional)
)
```

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `session_id` | `str` | `"default"` | Session identifier for scoping working memories |
| `db_path` | `str | None` | `None` | Path to the SQLite database file |

## Hermes Provider Config (config.yaml)

These keys live under `memory.mnemosyne` in Hermes Agent's `config.yaml`.

| Key | Default | Description |
|-----|---------|-------------|
| `auto_sleep` | `False` | Auto-run sleep() when working memory exceeds threshold |
| `sleep_threshold` | `50` | Working memory count before auto-sleep triggers |
| `vector_type` | `int8` | Vector storage type (float32, int8, bit) |
| `ignore_patterns` | `[]` | Regex patterns to filter from memory storage |
| `profile_isolation` | `False` | Per-profile memory isolation via Mnemosyne banks |
| `shared_surface_path` | `data/shared/mnemosyne.db` | SQLite path for shared surface memories |
| `shared_surface_read` | `False` | Merge shared-surface results into private bank recall |
| `skip_contexts` | `cron,flush,subagent,background,skill_loop` | Agent contexts to skip Mnemosyne init |
| `sync_roles` | `['user', 'assistant']` | Conversation roles to autosave in sync_turn() |

### Embeddings & Vector Search

Mnemosyne uses **BAAI/bge-small-en-v1.5** via fastembed (384 dimensions) for local embedding generation. No external API key is needed.

| Variable | Default | Description |
|----------|---------|-------------|
| `MNEMOSYNE_VEC_TYPE` | `int8` | Vector storage format (int8, float32, float16, binary) |
| `MNEMOSYNE_EMBEDDING_MODEL` | `BAAI/bge-small-en-v1.5` | fastembed model for vector embeddings |
| `MNEMOSYNE_EMBEDDING_DIM` | `384` | Override embedding vector dimension |
| `MNEMOSYNE_EMBEDDING_API_KEY` | â€” | API key for cloud embedding provider |
| `MNEMOSYNE_EMBEDDING_API_URL` | `https://openrouter.ai/api/v1` | API endpoint for cloud embeddings |
| `MNEMOSYNE_NO_EMBEDDINGS` | `false` | Disable dense vector retrieval entirely |
| `MNEMOSYNE_EMBEDDINGS_VIA_API` | `false` | Force cloud API mode for embeddings |
| `MNEMOSYNE_EMBEDDING_FALLBACK_MODEL` | â€” | Local fastembed model for API fallback (v3.12.0) |
