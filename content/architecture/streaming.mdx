# Streaming & Delta Sync

Real-time memory event streaming and incremental synchronization for distributed agent setups.

## MemoryStream

Thread-safe event stream with push (callbacks) and pull (iterator) patterns.

<CodeBlock code={`from mnemosyne import Mnemosyne

mem = Mnemosyne()
stream = mem.stream

mem.remember("test memory")

# Pull via callback
stream.on(lambda event: print(f"Event: {event.type}"))

# Pull via iterator
stream.listen()
`} language="python" />

## DeltaSync

Checkpoint-based incremental synchronization.

<CodeBlock code={`mem_a = Mnemosyne(session_id="agent-a")
mem_b = Mnemosyne(session_id="agent-b")

# Compute and apply delta
delta = mem_a.delta_sync.compute_delta(peer_id="agent-b")
mem_b.delta_sync.apply_delta(delta)

# Bidirectional sync
mem_a.delta_sync.sync_to("agent-b")
mem_a.delta_sync.sync_from(mem_b)
`} language="python" />

<Callout type="info">
DeltaSync uses memory IDs and timestamps for minimal change sets. Only new or modified memories transfer.
</Callout>
