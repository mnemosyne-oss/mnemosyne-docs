# Quick Start

Get Mnemosyne v3.7.0 running in under 5 minutes. This guide covers installation, basic configuration, and your first memory operations — including entity extraction and temporal recall.

## Prerequisites

- Python 3.10 or higher
- pip or uv package manager

## Install

Install Mnemosyne **with semantic search**:

<CodeBlock code={`# Recommended: pipx (isolated, survives env rebuilds)
pipx install mnemosyne-memory[embeddings]

# Or traditional pip
pip install mnemosyne-memory[embeddings]

# Using uv (faster)
uv pip install mnemosyne-memory[embeddings]

# Upgrade
pip install --upgrade mnemosyne-memory[embeddings]
# or: pipx upgrade mnemosyne-memory[embeddings]
`} language="bash" />

<Callout type="info" title="What [embeddings] gives you">
The `[embeddings]` (now default) installs **fastembed**, which powers semantic vector search. With it, Mnemosyne can:

- Match memories by **meaning**, not just keywords
- Find "Italian food" when you search "pizza" (semantic similarity)
- Combine vector + text scores for **hybrid retrieval**

Embeddings (fastembed + sqlite-vec) are included by default when using the Hermes plugin. For standalone Python SDK, add `[embeddings]` for vector search.
</Callout>

## Your First Memories

<CodeBlock code={`from mnemosyne import Mnemosyne

mem = Mnemosyne()

# Store a memory with entity extraction
mem.remember(
    "Alice prefers dark mode for the UI. She's the project lead.",
    extract_entities=True,
    importance=0.8,
)

# Store with LLM fact extraction
mem.remember(
    "The deadline was moved to June 15th.",
    extract=True,
)

# Temporal recall — find what happened recently
results = mem.recall("UI preferences", top_k=5)

for r in results:
    print(f"[{r['score']:.3f}] {r['content']}")
`} language="python" />

## Configurable Retrieval

Mnemosyne v3 lets you tune scoring weights per query:

<CodeBlock code={`# Boost text matching over semantic similarity
results = mem.recall(
    "exact error code E501",
    vec_weight=20.0,
    fts_weight=60.0,
    importance_weight=20.0,
)

# Emphasize very recent memories
results = mem.recall(
    "what did we discuss today?",
    temporal_weight=0.6,
    temporal_halflife=24.0,  # 24-hour half-life
)
`} language="python" />

## Memory Banks

Isolate memories for different projects or contexts:

<CodeBlock code={`# Create a bank for work projects
work_mem = Mnemosyne(bank="work")
work_mem.remember("Deployed v3.7.0 to staging.")

# Separate bank for personal
personal_mem = Mnemosyne(bank="personal")
personal_mem.remember("Dentist appointment on Friday.")
`} language="python" />

## Optional: sqlite-vec for Performance

`sqlite-vec` is bundled inside the Hermes plugin. Standalone pip users need `[embeddings]`. It moves vector similarity search into native C code inside SQLite. Without it, Mnemosyne uses an in-memory numpy fallback that works fine for most use cases.

<CodeBlock code={`pip install sqlite-vec`} language="bash" />

## Verify Installation

Confirm everything is working:

<CodeBlock code={`python -c "import mnemosyne; print(mnemosyne.__version__)"
# Should print: 3.7.0
`} language="bash" />
