# First Steps

A hands-on walkthrough of Mnemosyne's core operations.

## 1. Initialize Memory

<CodeBlock code={`from mnemosyne import Memory

mem = Memory()
print(f"Initialized: {mem.db_path}")
`} language="python" />

## 2. Store Information

<CodeBlock code={`# Simple memory
mem.remember("Alice is a backend engineer who prefers Go over Python.")

# Structured memory with metadata
mem.remember(
    content="Project deadline moved to Friday.",
    tags=["project-alpha", "deadline"],
    importance=0.8,
    source="slack",
)
`} language="python" />

## 3. Recall Information

<CodeBlock code={`# Semantic search
results = mem.recall("What programming language does Alice prefer?")

# Filtered search
results = mem.recall(
    query="deadline",
    tags=["project-alpha"],
    since="2026-04-01",
)

# Access a specific memory by ID
memory = mem.get("mem_abc123")
`} language="python" />

## 4. Update and Delete

<CodeBlock code={`# Update a memory
mem.update("mem_abc123", content="Deadline moved to Monday.")

# Delete a memory
mem.forget("mem_abc123")

# Bulk delete by tag
mem.forget_all(tags=["temp"])
`} language="python" />

## 5. Inspect Memory State

<CodeBlock code={`# Statistics
stats = mem.stats()
print(f"Working: {stats.working_count}")
print(f"Episodic: {stats.episodic_count}")
print(f"Database size: {stats.db_size_mb:.1f} MB")
`} language="python" />
