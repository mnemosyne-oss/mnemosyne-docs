# First Steps

A hands-on walkthrough of Mnemosyne's core operations.

## 1. Initialize Memory

<CodeBlock code={`from mnemosyne import Mnemosyne

mem = Mnemosyne()
// Or with a named session and multi-agent identity (v2.1):
mem = Mnemosyne(
    session_id="agent-42",
    author_id="alice",
    author_type="human",
    channel_id="team-alpha",
)
`} language="python" />

## 2. Store Information

<CodeBlock code={`// Simple memory
mem.remember("Alice is a backend engineer who prefers Go over Python.")

// With metadata, importance, and veracity confidence (v2.3)
mem.remember(
    content="Project deadline moved to Friday.",
    source="slack",
    importance=0.8,
    metadata={"project": "alpha"},
    veracity="stated",  // explicitly-stated fact
)

// With entity extraction — captures "PostgreSQL" as a triple
mem.remember("We decided to use PostgreSQL.", extract_entities=True)
`} language="python" />

## 3. Recall Information

<CodeBlock code={`# Semantic search
results = mem.recall("What programming language does Alice prefer?")

# Limit the number of results
results = mem.recall("deadline", top_k=3)

# Get recent context for a prompt
context = mem.get_context(limit=5)
`} language="python" />

## 4. Update and Delete

<CodeBlock code={`# Update a memory's content or importance
mem.update("mem_abc123", content="Deadline moved to Monday.")

# Delete a single memory
mem.forget("mem_abc123")

# Mark a memory as superseded by a newer one
mem.invalidate("mem_old456", replacement_id="mem_new789")
`} language="python" />

## 5. Inspect Memory State

<CodeBlock code={`# Get statistics
stats = mem.get_stats()
print(f"Working: {stats['working_count']}")
print(f"Episodic: {stats['episodic_count']}")
print(f"Database size: {stats['db_size_mb']:.1f} MB")
`} language="python" />

## 6. Use the Scratchpad

<CodeBlock code={`# Jot down transient notes
note_id = mem.scratchpad_write("Remember to check the API logs tomorrow.")

# Read scratchpad entries
notes = mem.scratchpad_read()

# Clear when done
mem.scratchpad_clear()
`} language="python" />
