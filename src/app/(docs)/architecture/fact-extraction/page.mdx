# Fact Extraction

Mnemosyne v2 can extract structured facts from memory content using an LLM, storing them as triples for knowledge-graph queries.

## Fallback Chain

1. **Remote OpenAI-compatible API** - fastest, best quality
2. **Local ctransformers GGUF** - offline, no API key
3. **Skip** - graceful degradation, never raises

<Callout type="info">
Fact extraction is **opt-in**. Pass `extract=True` to `remember()`.
</Callout>

## Usage

<CodeBlock code={`from mnemosyne import Mnemosyne

mem = Mnemosyne()
mid = mem.remember(
    "Mnemosyne uses SQLite for storage and fastembed for embeddings",
    importance=0.8,
    extract=True
)
`} language="python" />

## Standalone Usage

<CodeBlock code={`from mnemosyne.core.extraction import extract_facts, extract_facts_safe

facts = extract_facts("The project deadline is June 15th.")
facts = extract_facts_safe("Always safe, never raises.")
`} language="python" />

## Configuration

| Variable | Description |
|----------|-------------|
| `MNEMOSYNE_LLM_URL` | OpenAI-compatible API endpoint |
| `MNEMOSYNE_LLM_KEY` | API key |
| `MNEMOSYNE_LLM_MODEL` | Model name |
