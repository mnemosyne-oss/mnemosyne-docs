# MCP Server

MCP (Model Context Protocol) server exposing memory operations as tools for Claude Desktop, Hermes, and any MCP-compatible agent.

## Quick Start

<CodeBlock code={`# stdio transport (Claude Desktop, etc.)
mnemosyne mcp

# SSE transport (web clients)
mnemosyne mcp --transport sse --port 8080

# Scoped to a bank
mnemosyne mcp --bank project_a
`} language="bash" />

## Tools

The MCP server exposes all 24 Mnemosyne tools:

| Tool | Description |
|------|-------------|
| `mnemosyne_remember` | Store a memory |
| `mnemosyne_recall` | Search memories |
| `mnemosyne_sleep` | Run consolidation |
| `mnemosyne_stats` | Memory statistics |
| `mnemosyne_get` | Retrieve a single memory by ID |
| `mnemosyne_update` | Update a memory's content or importance |
| `mnemosyne_forget` | Permanently delete a memory |
| `mnemosyne_invalidate` | Mark a memory as expired |
| `mnemosyne_validate` | Attest, update, or invalidate a memory |
| `mnemosyne_export` | Export all memories to JSON |
| `mnemosyne_import` | Import memories from JSON or another provider |
| `mnemosyne_diagnose` | Run PII-safe diagnostics |
| `mnemosyne_shared_remember` | Store cross-agent surface memory |
| `mnemosyne_shared_recall` | Search shared surface memory |
| `mnemosyne_shared_forget` | Delete shared surface memory |
| `mnemosyne_shared_stats` | Shared surface DB stats |
| `mnemosyne_triple_add` | Add a temporal fact triple |
| `mnemosyne_triple_end` | End a temporal fact triple |
| `mnemosyne_triple_query` | Query the temporal knowledge graph |
| `mnemosyne_scratchpad_write` | Write to scratchpad |
| `mnemosyne_scratchpad_read` | Read scratchpad |
| `mnemosyne_scratchpad_clear` | Clear scratchpad |
| `mnemosyne_graph_query` | Traverse the memory graph |
| `mnemosyne_graph_link` | Link two memories with an edge |

## Claude Desktop Config

```json
{
  "mcpServers": {
    "mnemosyne": {
      "command": "mnemosyne",
      "args": ["mcp"]
    }
  }
}
```

<Callout type="info">
MCP is an open standard. Any MCP-compatible agent can connect without custom integration code.
</Callout>
