### remember

Store a memory in Working Memory.

<CodeBlock code={`remember(
    content: str,

    source: str = "user",
    session_id: str | None = None,

`} language="python" />

### recall

Retrieve relevant memories using hybrid search.

<CodeBlock code={`recall(
    query: str,
    top_k: int = 10,
    tags: list[str] | None = None,
    since: str | None = None,
    vector_weight: float = 0.5,
    text_weight: float = 0.3,
    temporal_weight: float = 0.2,
) -&gt; list[MemoryResult]
`} language="python" />

## Exceptions

| Exception | Raised When |
|-----------|-------------|
| `MemoryNotFoundError` | ID does not exist |
| `DatabaseError` | SQLite operation fails |
| `EmbeddingError` | Embedding generation fails |
| `ConfigurationError` | Invalid config or env vars |
| `EncryptionError` | Encryption/decryption fails |
