# Temporal Recall

Time-aware scoring that boosts recent memories over older ones using exponential decay.

## Formula

```
boost = exp(-hours_since_memory / halflife)
final_score = score * (1.0 + temporal_weight * boost)
```

## Usage

<CodeBlock code={`from mnemosyne import Mnemosyne

mem = Mnemosyne()

results = mem.recall(
    "tasks",
    temporal_weight=0.5,
    temporal_halflife=12.0,
    query_time="2026-04-29T12:00:00"
)
`} language="python" />

## Parameters

| Parameter | Default | Description |
|-----------|---------|-------------|
| `temporal_weight` | 0.0 | 0.0 = off, 1.0 = strong |
| `query_time` | now | Reference time for decay |
| `temporal_halflife` | 24.0 | Hours for 50% decay |

<Callout type="info">
Applied across all recall tiers: working, episodic, entity, and fact matches.
</Callout>

## Examples

<CodeBlock code={`# Strong recency bias
results = mem.recall("current task", temporal_weight=0.8)

# Historical query
results = mem.recall("decisions", temporal_weight=0.3,
    query_time="2026-04-01T00:00:00")

# Fast-changing context
results = mem.recall("messages", temporal_weight=0.6,
    temporal_halflife=2.0)
`} language="python" />
