# Configurable Scoring

Tune the hybrid scoring formula per query, adjusting vector similarity, keyword matching, and importance.

## Formula

```
score = vec_weight * vector_sim + fts_weight * fts_rank + importance_weight * importance
```

Weights auto-normalize to sum to 1.0.

## Defaults

| Component | Weight | Role |
|-----------|--------|------|
| Vector | 50% | Semantic matching |
| FTS5 | 30% | Exact term matching |
| Importance | 20% | Stored priority |

## Per-Query Overrides

<CodeBlock code={`from mnemosyne import Mnemosyne
mem = Mnemosyne()

# Semantic emphasis
results = mem.recall("code patterns",
    vec_weight=0.7, fts_weight=0.2, importance_weight=0.1)

# Keyword emphasis
results = mem.recall("exact error message",
    vec_weight=0.2, fts_weight=0.7, importance_weight=0.1)

# Combined with temporal
results = mem.recall("recent discussions",
    temporal_weight=0.5,
    vec_weight=0.4, fts_weight=0.3, importance_weight=0.3)
`} language="python" />

## Environment Variables

| Variable | Default |
|----------|---------|
| `MNEMOSYNE_VEC_WEIGHT` | 0.5 |
| `MNEMOSYNE_FTS_WEIGHT` | 0.3 |
| `MNEMOSYNE_IMPORTANCE_WEIGHT` | 0.2 |
