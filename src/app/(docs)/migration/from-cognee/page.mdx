# From Cognee

Import all memories from [Cognee](https://www.cognee.ai) into Mnemosyne. Cognee stores memories as a graph (Kuzu + LanceDB + SQLite). The importer converts nodes to episodic memories and edges to knowledge graph triples.

## One Command

```bash
hermes mnemosyne import --from cognee
```

## What Gets Imported

- **Graph nodes**: Each Cognee node becomes an episodic memory, enriched with its properties
- **Graph edges**: `subject → predicate → object` edges become TripleStore triples
- **Node types**: Mapped to `author_type` for multi-agent differentiation
- **Dataset identity**: Cognee dataset ID mapped to `channel_id`
- **Properties**: Node properties merged into Mnemosyne metadata
- **Timestamps**: Preserved where available

## Options

```bash
# Dry run first to see what will be imported
hermes mnemosyne import --from cognee --dry-run

# Assign to a specific channel
hermes mnemosyne import --from cognee --channel-id my-dataset

# Import from a custom data directory
hermes mnemosyne import --from cognee --data-dir /path/to/cognee/data
```

## Extraction Method

The Cognee importer uses `get_graph_data()` to pull the full graph — nodes become episodic memories, edges become triples. The graph structure is preserved so no information is lost.

## Identity Mapping

| Cognee Field | Mnemosyne Field |
|-------------|----------------|
| Node content | `content` |
| Node type | `author_type` |
| Node properties | `metadata` |
| Edge (subject → predicate → object) | `triple_add(subject, predicate, object)` |
| Dataset ID | `channel_id` |

## Programmatic Import

```python
from mnemosyne import Mnemosyne
from mnemosyne.core.importers import import_from_cognee

mnemosyne = Mnemosyne(session_id="migration")

result = import_from_cognee(
    mnemosyne=mnemosyne,
    data_dir="/path/to/cognee/data",   # optional
    channel_id="my-dataset",            # optional
    dry_run=True,                       # try True first
)

print(f"Imported {result.imported} of {result.total}")
print(f"Nodes: {result.nodes}, Edges: {result.edges}")
```
