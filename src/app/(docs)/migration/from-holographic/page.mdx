# From Holographic Memory

Import facts from [Hermes Holographic Memory](https://github.com/NousResearch/hermes-agent/tree/main/plugins/memory/holographic) into Mnemosyne. The Holographic Memory plugin is a local-first SQLite fact store for Hermes Agent using FTS5 search, HRR (Holographic Reduced Representations) for compositional queries, entity linking, and trust scoring.

## One Command

```bash
hermes mnemosyne import --from holographic
```

The importer reads directly from the default location (`~/.hermes/memory_store.db`).

## Custom Path & Filters

```bash
# Specify a custom database path
hermes mnemosyne import --from holographic --db-path /path/to/memory_store.db

# Only import high-trust facts
hermes mnemosyne import --from holographic --min-trust 0.5

# Filter by category
hermes mnemosyne import --from holographic --category-filter preferences

# Dry run first
hermes mnemosyne import --from holographic --dry-run
```

## What Gets Preserved

| Holographic Field | Mnemosyne Destination |
|-------------------|-----------------------|
| `content` | Memory content (verbatim) |
| `trust_score` | `importance` (direct 0.0–1.0 mapping) |
| `category` | `metadata._holographic_category` |
| `tags` | `metadata._holographic_tags` |
| `created_at` | `metadata._created_at` (original timestamp) |
| `updated_at` | `metadata._updated_at` |
| `retrieval_count` | `metadata._holographic_retrieval_count` |
| `helpful_count` | `metadata._holographic_helpful_count` |
| Entity names | Entity annotations for entity-aware recall |

## Python API

```python
from mnemosyne.core.importers import HolographicImporter, import_from_holographic

# From default location
importer = HolographicImporter()
result = importer.run(mnemosyne)

# Custom path with filters
importer = HolographicImporter(
    db_path="~/.hermes/memory_store.db",
    min_trust=0.3,
    category_filter="technical",
)
result = importer.run(mnemosyne, dry_run=True)

# Convenience wrapper
result = import_from_holographic(mnemosyne, db_path="~/.hermes/memory_store.db")
```

## What's Not Imported

- **HRR vectors** — Mnemosyne uses its own embedding engine (sqlite-vec + configurable models) for vector search. The Holographic HRR BLOBs are provider-specific and not transferable.
- **memory_banks** — These are derived bundled HRR vectors. Category names are preserved in metadata for reference.

## Result Object

```python
result.provider      # "holographic"
result.total         # Total items found
result.imported      # Successfully inserted
result.failed        # Insertion errors
result.memory_ids    # List of imported memory IDs
result.errors        # List of error strings
```
