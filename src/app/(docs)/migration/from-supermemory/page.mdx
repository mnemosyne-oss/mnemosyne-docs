# From SuperMemory

Import all memories from [SuperMemory](https://supermemory.ai) into Mnemosyne. SuperMemory is a cloud-based memory platform with document storage and search. The importer pulls all documents and search results.

## One Command

```bash
hermes mnemosyne import --from supermemory --api-key sk-xxx
```

## What Gets Imported

- **Documents**: All stored documents via `documents.list()` pagination
- **Search results**: Results from `search.execute()` for key queries
- **Container tags**: SuperMemory container tags mapped to `channel_id`
- **Document metadata**: Original metadata and timestamps preserved
- **Titles**: Document titles stored in metadata with `source="supermemory"`

## Options

```bash
# Dry run to preview
hermes mnemosyne import --from supermemory --api-key sk-xxx --dry-run

# Filter by container/tag
hermes mnemosyne import --from supermemory --api-key sk-xxx --container my-project

# Assign to a channel
hermes mnemosyne import --from supermemory --api-key sk-xxx --channel-id docs

# Include search results
hermes mnemosyne import --from supermemory --api-key sk-xxx --include-search
```

## Extraction Methods

1. **`documents.list()`** — Paginated listing of all stored documents
2. **`search.execute()`** — Results from semantic search (optional, use `--include-search`)

Both methods preserve original IDs and timestamps so nothing is lost.

## Identity Mapping

| SuperMemory Field | Mnemosyne Field |
|------------------|----------------|
| Document content | `content` |
| Document title | `metadata.title` |
| Container tags | `channel_id` |
| Document metadata | `metadata` |
| Created/updated | Preserved in metadata |
| Search results | `content` with `source="supermemory_search"` |

## Programmatic Import

```python
from mnemosyne import Mnemosyne
from mnemosyne.core.importers import import_from_supermemory

mnemosyne = Mnemosyne(session_id="migration")

result = import_from_supermemory(
    api_key="sk-xxx",
    mnemosyne=mnemosyne,
    container="my-project",         # optional filter
    channel_id="docs",              # optional
    dry_run=True,                   # try True first
)

print(f"Imported {result.imported} of {result.total}")
```
