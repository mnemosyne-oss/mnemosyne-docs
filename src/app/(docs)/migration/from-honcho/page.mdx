# From Honcho

Import all memories from [Honcho](https://honcho.dev) into Mnemosyne. Honcho uses a peer → session → message model backed by PostgreSQL + pgvector. The importer iterates every layer and preserves peer identity.

## One Command

```bash
hermes mnemosyne import --from honcho
```

## What Gets Imported

- **Session messages**: Each message becomes a working memory with `source="honcho_message"`
- **Session context**: `context()` output imported with `importance=0.7`
- **Peer identity**: Each peer's ID preserved as `author_id` (`honcho_peer:{peer_id}`)
- **Session metadata**: Original `session_id` preserved in metadata

## Options

```bash
# Dry run to preview
hermes mnemosyne import --from honcho --dry-run

# Filter by specific peer
hermes mnemosyne import --from honcho --peer-id alice

# Assign to a channel
hermes mnemosyne import --from honcho --channel-id my-team

# Custom Honcho API URL (self-hosted)
hermes mnemosyne import --from honcho --base-url http://localhost:8000
```

## Extraction Method

The Honcho importer iterates `peers → sessions → context() + messages` to extract everything. Each peer's identity is preserved so you can query by author after migration.

## SDK Dependencies

Install the Honcho SDK if using cloud:

```bash
pip install honcho-ai
```

Self-hosted instances work without the SDK using direct API calls.

## Identity Mapping

| Honcho Field | Mnemosyne Field |
|-------------|----------------|
| Message content | `content` |
| Peer ID | `author_id` (`honcho_peer:{peer_id}`) |
| Session context | `content` (importance 0.7) |
| User metadata | `metadata` |
| Session ID | Preserved in metadata |

## Programmatic Import

```python
from mnemosyne import Mnemosyne
from mnemosyne.core.importers import import_from_honcho

mnemosyne = Mnemosyne(session_id="migration")

result = import_from_honcho(
    mnemosyne=mnemosyne,
    peer_id="alice",                # optional filter
    channel_id="my-team",           # optional
    dry_run=True,                   # try True first
)

print(f"Imported {result.imported} of {result.total}")
print(f"Peers: {result.peer_count}, Messages: {result.message_count}")
```
