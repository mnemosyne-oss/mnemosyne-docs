# Memory Banks

Named namespace isolation where each bank gets its own SQLite database, completely independent from others.

## Usage

<CodeBlock code={`from mnemosyne import Mnemosyne
from mnemosyne.core.banks import BankManager

BankManager().create_bank("work")

work_mem = Mnemosyne(bank="work")
work_mem.remember("Client meeting at 3pm", importance=0.8)

default_mem = Mnemosyne()
default_mem.remember("General knowledge", importance=0.5)

# Completely independent
work_results = work_mem.recall("meeting")
default_results = default_mem.recall("meeting")  # Does NOT find it
`} language="python" />

## Management

<CodeBlock code={`from mnemosyne.core.banks import BankManager
manager = BankManager()

manager.create_bank("work")
banks = manager.list_banks()
exists = manager.bank_exists("work")
stats = manager.get_bank_stats("work")
manager.rename_bank("work", "work-v2")
manager.delete_bank("work")
`} language="python" />
